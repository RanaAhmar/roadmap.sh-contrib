## ✨ Overview

[roadmap.sh](https://roadmap.sh/) is the premier community-driven platform providing educational roadmaps, articles, and resources for developers. 

This repository serves as my personal workspace to research, draft, and format structural changes, new content, and translations before submitting official Pull Requests to the main `kamranahmedse/developer-roadmap` repository.

## 🗂️ What's Inside

- **`/drafts`**: Markdown outlines for proposed new roadmaps (e.g., AI Engineering, Developer Relations).
- **`/updates`**: Proposed node updates and resource links for existing roadmaps (Frontend, Backend, DevOps).
- **`/translations`**: Staging for i18n content updates.
- **`/scripts`**: Utility scripts to validate JSON structures against roadmap.sh schemas before PR submission.

## 🎯 Current Focus

Currently focusing on expanding the **AI/ML** and **System Design** roadmaps by integrating the latest industry standards surrounding Large Language Models and Agentic workflows.

## 🤝 Community

If you are interested in collaborating on a specific roadmap proposal before we submit it upstream, feel free to open an Issue here and let's discuss!
